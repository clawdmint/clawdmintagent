## Reveal Mekanizmasi

### Mint Oncesi (Pre-Reveal)
- Tum NFT'ler ayni "CLASSIFIED" placeholder SVG'yi gosterir
- Kilitli robot silueti + "CLASSIFIED" damgasi
- Metadata: tum trait'ler "???" olarak gorunur
- Kullanicilar ne alacagini bilmeden mint eder

### Mint Sonrasi (Sold Out = Reveal)
1. 10,000 SVG + metadata JSON zaten IPFS'te hazir bekler
2. Koleksiyon sold out olunca `setBaseURI()` cagrilir
3. Yeni IPFS hash ile tum NFT'ler reveal olur
4. Ardindan `freezeMetadata()` cagrilarak metadata kilitleme yapilir

**Not:** Mevcut `ClawdmintCollection.sol` contract'i `setBaseURI` ve `freezeMetadata` fonksiyonlarini zaten destekliyor. Ek contract degisikligi gerekmez.

## Teknik Uygulama

### SVG Uretim Pipeline'i
1. **Modular SVG parcalari** — Her trait icin ayri SVG snippet
2. **Generator script** (Node.js) — 10,000 benzersiz kombinasyon uretir
3. **Uniqueness check** — Ayni trait seti tekrarlanmaz
4. **Metadata JSON** — OpenSea/marketplace uyumlu ERC-721 metadata
5. **IPFS upload** — Pinata ile toplu yukleme
